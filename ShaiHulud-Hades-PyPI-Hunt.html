Executive Summary

The Shai-Hulud Hades wave is the latest iteration of a multi-month, multi-ecosystem supply-chain worm campaign operated by a cluster tracked as TeamPCP (aliases: PCPcat, DeadCatx3, ShellForce). On or about 2026-06-08, 37 malicious wheel (.whl) artifacts spanning 19 Python packages — predominantly bioinformatics and scientific computing tools such as dynamo-release, spateo-release, coolbox, ufish, and napari-ufish — were published to PyPI under legitimate maintainer accounts using stolen publishing tokens from an earlier campaign wave.

Each trojanized wheel drops a *-setup.pth file into the Python site-packages directory, silently triggering a multi-stage credential stealer on the next Python interpreter startup — no user interaction required beyond a standard pip install. The payload (_index.js, ~4.8 MB, obfuscated with ROT-N + AES-128-GCM) is executed by a quietly downloaded Bun v1.3.13 JavaScript runtime, deliberately avoiding Python-aware EDR detection. Post-exploitation includes harvesting of ~40+ credential categories (AWS, GCP, Azure, Kubernetes, GitHub, SSH, password managers, AI tool API keys), exfiltrating encrypted dumps to attacker-controlled GitHub repositories, establishing systemd / LaunchAgent persistence, injecting CI/CD workflow files, modifying IDE/AI tool configs (.claude/settings.json, .vscode/tasks.json), forging SLSA v1 provenance attestations on Sigstore, and self-propagating by publishing new trojanized packages using stolen registry tokens.

The campaign has compromised 471+ total artifacts across npm and PyPI. Any environment that installed an affected package version should treat all secrets on that system as fully compromised and begin immediate rotation. This pack covers two related "Hades" PyPI sub-waves: the primary 19-package / 37-wheel set reported by Socket and BleepingComputer (Section 3, Table A), and an earlier, sibling 6-package bioinformatics set documented by Endor Labs (Table B) that was quarantined within ~30 minutes of publication. Both use identical TTPs and the same TeamPCP infrastructure; several IOCs in this pack (the ensmallen_*.abi3.so binaries, the felixEvora C2 account, the AES-128 key, /tmp/.sshu-setup.js) originate specifically from the Endor 6-package wave and are labeled as such. Removed packages may still persist in mirrors, private indexes, or build caches.

⚠ Wiper risk: The malware checks GitHub organisation membership. If the victim is a member of targeted orgs the payload executes rm -rf ~/ and rm -rf ~/Documents before exfiltration. Isolate affected hosts before credential rotation.
