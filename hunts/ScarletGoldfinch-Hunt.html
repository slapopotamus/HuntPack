1. Executive Summary

Scarlet Goldfinch is a Windows initial-access activity cluster that uses compromised websites to convince users to execute malicious code. Red Canary reports that the cluster shifted in 2025 from fake browser-update lures toward paste-and-run, also known as ClickFix, with significant month-to-month changes in LOLBAS tradecraft.

The durable defender opportunity is not any single domain or command string. Build coverage around the repeated behaviors: user-context shell execution, web download cradles, LOLBAS proxy execution, script/archive staging, suspicious persistence, WMI-brokered execution, and NetSupport Manager or Remcos payload execution from user-writable paths.

Defender priority: Hunt for the execution chain first, then tune prevention around remote script download, mshta/msiexec/finger/forfiles abuse, and unsanctioned NetSupport Manager execution.

8. CrowdStrike LogScale CQL Hunt Queries

Schema note: These are Falcon LogScale / NG-SIEM starter hunts. Validate field names such as FileName, ImageFileName, CommandLine, ParentBaseFileName, DomainName, RemoteAddressIP4, RegistryKeyName, and TargetFileName in your tenant.

Q1. Curl Downloading Script-Like Payloads

CONF: High | FP: Medium | COST: Low | Requires: ProcessRollup2 command line

Finds curl commands that save downloaded content as script-capable or executable-adjacent file types, matching Scarlet Goldfinch's early paste-and-run BAT/script staging pattern.

#event_simpleName=ProcessRollup2
| FileName=/^curl(\.exe)?$/i
| CommandLine=/(\s-o\s|\s--output\s|\s--remote-name\s).*\.(bat|cmd|cpl|hta|ocx|scr|sys|vbs)(\"|\s|$)/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, CommandLine, ImageFileName, SHA256HashData], limit=200)

Q2. PowerShell IWR/IEX Download Cradle

CONF: Medium-High | FP: Medium | COST: Low | Requires: ProcessRollup2 command line

Looks for PowerShell web-download and execute patterns such as IWR, WebClient DownloadString, and IEX that Scarlet Goldfinch used to pull and run PS1 payload logic.

#event_simpleName=ProcessRollup2
| FileName=/^(powershell|powershell\.exe|pwsh|pwsh\.exe)$/i
| CommandLine=/(invoke-expression|\biex\b|downloadstring|new-object\s+system\.net\.webclient|invoke-webrequest|\biwr\b)/i
| CommandLine=/(http:\/\/|https:\/\/|\-nop|\-noprofile|\-w\s+hidden|\-windowstyle\s+hidden)/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, CommandLine, ImageFileName, SHA256HashData], limit=200)

Q3. Msiexec Remote Quiet Install

CONF: High | FP: Low-Medium | COST: Low | Requires: ProcessRollup2 command line

Flags msiexec pulling packages from HTTP or HTTPS with quiet install switches, covering the Scarlet Goldfinch epoch that used remote MSI execution to separate initial access from payload activity.

#event_simpleName=ProcessRollup2
| FileName=/^msiexec(\.exe)?$/i
| CommandLine=/(http:\/\/|https:\/\/)/i
| CommandLine=/(\s\/i\s|\s-i\s|\s\/package\s)/i
| CommandLine=/(\s\/q\b|\s\/qn\b|\s-q\b|\s-quiet\b)/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, CommandLine, ImageFileName], limit=200)

Q4. Mshta External URL Launcher

CONF: High | FP: Low-Medium | COST: Low | Requires: ProcessRollup2 command line

Catches mshta launching remote content, a high-signal LOLBAS behavior Scarlet Goldfinch used to fetch script content and hand execution to PowerShell.

#event_simpleName=ProcessRollup2
| FileName=/^mshta(\.exe)?$/i
| CommandLine=/(http:\/\/|https:\/\/)/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, CommandLine, ImageFileName], limit=200)

Q10. Cmd Delayed Expansion And Caret Obfuscation

CONF: Medium-High | FP: Medium | COST: Low | Requires: ProcessRollup2 command line

Finds command shells using delayed expansion, caret insertion, or substring extraction to hide curl, mshta, start, and related download/execute keywords, matching 2026 Scarlet Goldfinch/SmartApeSG obfuscation updates.

#event_simpleName=ProcessRollup2
| FileName=/^(cmd|cmd\.exe)$/i
| CommandLine=/\/v:on|![A-Za-z0-9_]+:~\d+,\d+!|\^[A-Za-z]\^[A-Za-z]\^[A-Za-z]/i
| CommandLine=/(curl|c\^u\^r\^l|mshta|m\^s\^h\^t\^a|start|s\^t\^a\^r\^t|http:\/\/|https:\/\/|\.hta)/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, CommandLine, ImageFileName], limit=200)

Q5. Finger Piped Into Command Shell

CONF: High | FP: Low | COST: Low | Requires: ProcessRollup2 command line

Hunts rare finger.exe abuse where output from a remote Finger server is piped into cmd, matching Scarlet Goldfinch's December paste-and-run pivot.

#event_simpleName=ProcessRollup2
| (FileName=/^finger(\.exe)?$/i OR CommandLine=/\bfinger\s+\S+@\d{1,3}(\.\d{1,3}){3}/i OR CommandLine=/\bfinger\s+\S+@\S+\s*\|\s*cmd/i)
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, CommandLine, ImageFileName], limit=200)

Q6. Forfiles Proxy To Cmd Or Mshta

CONF: High | FP: Low-Medium | COST: Low | Requires: ProcessRollup2 command line

Identifies forfiles.exe being used as an indirect launcher for cmd, PowerShell, mshta, curl, or URLs, covering the later Scarlet Goldfinch proxy-execution chain.

#event_simpleName=ProcessRollup2
| FileName=/^forfiles(\.exe)?$/i
| CommandLine=/\/c\s+.*(cmd|powershell|mshta|curl|http:\/\/|https:\/\/)/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, CommandLine, ImageFileName], limit=200)

Q7. WmiPrvSE Spawns User-Writable Payload

CONF: Medium-High | FP: Medium | COST: Low | Requires: ProcessRollup2 ancestry and image path

Looks for WMI-brokered execution where WmiPrvSE launches an executable from ProgramData, AppData, Public, Temp, or Downloads, useful for finding Invoke-CimMethod process-tree breaks and DLL side-load follow-on payloads.

#event_simpleName=ProcessRollup2
| ParentBaseFileName=/^wmiprvse\.exe$/i
| ImageFileName=/\\(ProgramData|Users\\[^\\]+\\AppData\\|Users\\Public\\|Temp\\|Downloads\\)/i
| FileName=/\.(exe|dll)$/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, ImageFileName, CommandLine, MD5HashData, SHA256HashData], limit=200)

Q8. Persistence Paths Used In Scarlet Goldfinch Chains

CONF: Medium | FP: Medium | COST: Medium | Requires: registry, file-write, scheduled-task process events

Correlates persistence-related registry, Startup folder, scheduled task, and file-write signals tied to HKCU Run, UserInitMprLogonScript, Startup LNKs, and user-writable payload paths.

(#event_simpleName=RegistryModification OR #event_simpleName=NewExecutableWritten OR #event_simpleName=ProcessRollup2)
| (CommandLine=/UserInitMprLogonScript|\\CurrentVersion\\Run|schtasks\s+\/create|Startup\\.*\.lnk/i OR RegistryKeyName=/UserInitMprLogonScript|\\CurrentVersion\\Run/i OR TargetFileName=/\\Startup\\.*\.lnk$/i)
| (CommandLine=/\\(ProgramData|Users\\[^\\]+\\AppData\\|Users\\Public\\)/i OR TargetFileName=/\\(ProgramData|Users\\[^\\]+\\AppData\\|Users\\Public\\)/i OR RegistryData=/\\(ProgramData|Users\\[^\\]+\\AppData\\|Users\\Public\\)/i)
| table([@timestamp, ComputerName, UserName, #event_simpleName, FileName, CommandLine, RegistryKeyName, RegistryData, TargetFileName], limit=200)

Q9. NetSupport Client From Non-Standard Paths

CONF: Medium-High | FP: Medium | COST: Low | Requires: process path and RMM allowlist context

Surfaces NetSupport Manager client execution from suspicious or user-writable directories so analysts can separate sanctioned RMM use from Scarlet Goldfinch payload deployment.

#event_simpleName=ProcessRollup2
| (FileName=/^client32\.exe$/i OR CommandLine=/\\(client32|ntcache|spcache)\.exe/i)
| ImageFileName=/\\(ProgramData|Users\\[^\\]+\\AppData\\|Users\\Public\\|Temp\\|Downloads\\)/i
| table([@timestamp, ComputerName, UserName, ParentBaseFileName, FileName, ImageFileName, CommandLine, MD5HashData, SHA256HashData], limit=200)

12. Hardening - Tiered & Deployable

Immediate

  • Baseline sanctioned RMM tools and flag NetSupport Manager outside approved hosts and paths.
  • Block or alert on mshta making internet requests where business use is absent.
  • Alert on finger.exe network use and finger piped to cmd on workstations.
  • Review proxy and DNS logs for the historical domains and IPs in the appendix.

Near Term

  • Enable PowerShell script block and module logging for workstations and admin hosts.
  • Deploy ASR or application-control policy for script and LOLBAS abuse where compatible.
  • Monitor Startup folder LNK creation and HKCU Environment UserInitMprLogonScript writes.
  • Require administrative review for new remote support software or trialware.

Strategic

  • Adopt WDAC or AppLocker policy for high-risk user groups and unmanaged endpoints.
  • Build paste-and-run detection as a behavior family rather than one indicator-specific rule.
  • Continuously compare RMM telemetry against CMDB or endpoint-management inventory.
  • Train users and help desk teams on fake CAPTCHA and ClickFix prompts that request Run dialog commands.

PowerShell: Quick Local Persistence Sweep

Endpoint triage helper; run in an approved response shell.
$paths = @(
  "HKCU:\Software\Microsoft\Windows\CurrentVersion\Run",
  "HKCU:\Environment"
)

foreach ($path in $paths) {
  if (Test-Path $path) {
    Get-ItemProperty -Path $path | Select-Object PSPath, *
  }
}

$startup = Join-Path $env:APPDATA "Microsoft\Windows\Start Menu\Programs\Startup"
if (Test-Path $startup) {
  Get-ChildItem -Path $startup -Force | Select-Object FullName, Length, LastWriteTime
}
